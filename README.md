# AI Workflow Seminar — An Informal Introduction to Claude Code

An informal introduction to [Claude Code](https://docs.anthropic.com/en/docs/claude-code) for researchers. We'll look at what Claude Code is, how it works, and try it out on some real tasks.

This repository has the slides, example materials, and a hackathon handout.

---

## Prerequisites

- A computer with **Claude Code installed** — see the [quickstart guide](https://code.claude.com/docs/en/quickstart)

---

## Setup

Clone this repository:

```bash
git clone https://github.com/coadywing/ai_workflow_seminar.git
cd ai_workflow_seminar
```

Start Claude Code:

```bash
claude
```
